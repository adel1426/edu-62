# 🌸 منصة الثانية والستون التعليمية — رفع الموقع على Hostinger

دليل بسيط جداً، خطوة بخطوة 💖

---

## 📋 ما تحتاجين قبل البدء:

- حساب Hostinger مفعّل (الذي اشتريتيه)
- اسم النطاق (الدومين) الذي تريدين استخدامه
- هذا المجلد كاملاً (`hostinger-version/`)

---

## 🚀 الخطوات (15 دقيقة فقط):

### 1️⃣ إنشاء قاعدة البيانات في Hostinger

1. ادخلي إلى **hpanel.hostinger.com**
2. اختاري الموقع → **Databases → Management** (إدارة قواعد البيانات)
3. اضغطي **Create New Database** (إنشاء قاعدة بيانات جديدة)
4. اكتبي:
   - **Database name:** `nour` (سيصبح اسمها مثلاً `u123456_nour`)
   - **Username:** `nour` (سيصبح مثلاً `u123456_nour`)
   - **Password:** اختاري كلمة قوية واحفظيها 🔑
5. اضغطي **Create**
6. **انسخي 4 معلومات** ستظهر في الجدول:
   - Database name: مثلاً `u123456789_nour`
   - Username: مثلاً `u123456789_nour`
   - Password: التي اخترتيها
   - Host: غالباً `localhost`

---

### 2️⃣ تعديل ملف الإعدادات

1. افتحي ملف **`api/config.php`** بأي محرر نصوص (Notepad أو VS Code)
2. ابحثي عن هذه الأسطر وعدّليها:

```php
define('DB_NAME',     'u123456789_nour');     // ← ضعي اسم قاعدة بياناتك
define('DB_USER',     'u123456789_nour');     // ← ضعي اسم المستخدم
define('DB_PASS',     'Your_Password_Here');  // ← ضعي كلمة المرور
```

3. احفظي الملف ✅

> 💡 **اختياري:** إذا أردت تغيير كلمة مرور لوحة الإدارة، عدّلي `ADMIN_PASSWORD` في نفس الملف.

---

### 3️⃣ رفع الملفات إلى الموقع

1. في hpanel، اذهبي إلى **Files → File Manager** (مدير الملفات)
2. ادخلي إلى مجلد **`public_html`** (احذفي أي ملفات افتراضية فيه)
3. اضغطي زر **Upload** (رفع)
4. ارفعي **كل محتوى مجلد** `hostinger-version` (وليس المجلد نفسه):
   - `index.html`
   - `admin.html`
   - `install.php`
   - `data.sql`
   - `.htaccess`
   - مجلد `api/` كاملاً

> 💡 الأسهل: اضغطي محتوى `hostinger-version` في ملف ZIP، ارفعيه، ثم استخدمي **Extract** داخل File Manager.

---

### 4️⃣ تشغيل التثبيت بضغطة واحدة

1. افتحي المتصفح واذهبي إلى:
   ```
   https://yourdomain.com/install.php
   ```
   (استبدلي `yourdomain.com` باسم موقعك)

2. ستفحص الصفحة الاتصال بقاعدة البيانات تلقائياً
3. اضغطي زر **▶️ بدء التثبيت**
4. ستظهر رسائل نجاح خضراء ✅:
   - جدول الأسئلة جاهز
   - جدول النتائج جاهز
   - جدول محتوى الدروس جاهز
   - تم إدخال 168 سؤال

---

### 5️⃣ ⚠️ احذفي ملفي التثبيت (مهم للأمان!)

1. ارجعي إلى **File Manager**
2. احذفي:
   - `install.php`
   - `data.sql`

---

### 6️⃣ زوري موقعك! 🎉

- **الموقع الرئيسي:** `https://yourdomain.com`
- **لوحة الإدارة:** `https://yourdomain.com/admin.html`
  - اسم المستخدم: `admin`
  - كلمة المرور: `62`

---

## ❓ مشاكل شائعة وحلولها:

### ❌ "فشل الاتصال بقاعدة البيانات"
- تأكدي أن `api/config.php` فيه القيم الصحيحة من Hostinger
- تأكدي أن قاعدة البيانات أُنشئت فعلاً

### ❌ صفحة بيضاء أو خطأ 500
- في File Manager، تأكدي أن ملف `.htaccess` موجود في `public_html` و في `api/`
- إذا لم تظهر الملفات المخفية: من زر الإعدادات في File Manager، فعّلي **Show Hidden Files**

### ❌ "Route not found" عند فتح الموقع
- معناها أن `.htaccess` لا يعمل
- تواصلي مع دعم Hostinger واطلبي تفعيل `mod_rewrite` (يكون مفعلاً افتراضياً)

### ❌ الصور أو الأسئلة لا تظهر
- افتحي لوحة الإدارة وتحققي من عدد الأسئلة المعروضة
- إذا كانت 0، شغّلي `install.php` مرة أخرى

---

## 📞 ملاحظات مهمة:

- **النسخ الاحتياطية:** كل أسبوع، اذهبي إلى Hostinger → Files → Backup لعمل نسخة كاملة
- **شهادة SSL مجانية:** فعّليها من Hostinger → SSL → Setup (تجعل الموقع `https`)
- **بريد الموقع:** يمكنك إنشاء بريد `info@yourdomain.com` من Hostinger → Emails
